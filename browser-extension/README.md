# 红墨剪藏（RedInk 浏览器插件）

在小红书 / 抖音的网页版笔记页一键剪藏标题、作者、正文、话题标签与互动数据，发送到本地
RedInk 的「剪藏收件箱」，然后在 RedInk 的「对标拆解」页一键拆解爆款套路。

- 纯原生 JS（Manifest V3），无构建步骤、无第三方依赖，目录即插件
- 数据只发到你本机的 RedInk（默认 `http://127.0.0.1:12398`），不经过任何第三方服务器
- 使用你浏览器里自己的登录态浏览页面，插件不读取、不上传任何账号凭证

## 安装（Chrome / Edge）

1. 打开扩展管理页：
   - Chrome 地址栏输入 `chrome://extensions`
   - Edge 地址栏输入 `edge://extensions`
2. 打开右上角（Edge 为左侧）的「开发者模式」开关
3. 点「加载已解压的扩展程序」，选择本目录（`browser-extension/`）
4. 工具栏出现红底白字「红」图标即安装成功（建议固定到工具栏）

## 使用

1. 启动本地 RedInk（后端监听 `127.0.0.1:12398`）
2. 在浏览器打开一篇小红书或抖音的笔记/视频页（**网页版**，不是 App 内分享卡片）：
   - 小红书：`https://www.xiaohongshu.com/explore/<笔记 id>` 等笔记详情页
   - 抖音：`https://www.douyin.com/video/<视频 id>`、`/note/<图文 id>` 等
3. 点击工具栏的「红」图标，弹窗会显示提取到的标题与正文预览
4. 确认无误后点「发送到 RedInk」
5. 打开 RedInk 的「对标拆解」页，在「剪藏收件箱」里点「拆解这条」即可

## 支持站点

| 站点 | 提取内容 |
| --- | --- |
| 小红书（www.xiaohongshu.com） | 标题、作者、正文、话题标签、点赞/收藏/评论数（能取则取） |
| 抖音（www.douyin.com） | 标题/文案、作者、话题标签、互动数（能取则取） |
| 其他站点 | 仅兜底提取网页标题与 meta 描述 |

> 说明：小红书/抖音页面结构频繁改版，个别字段（尤其互动数）可能临时取不到。
> 标题与正文有多级兜底（meta 描述、网页标题清洗），若完全提取失败，
> 请等页面加载完成后重试；仍失败说明平台改版，需更新 `content.js` 顶部的候选选择器。

## 设置

点开弹窗底部的「设置」：

- **RedInk 端口**：默认 `12398`，与本地 RedInk 的端口保持一致即可
- **访问令牌（选填）**：如果你启动 RedInk 时设置了环境变量 `REDINK_ACCESS_TOKEN`，
  把同一个令牌填在这里，插件会以 `X-Access-Token` 请求头随剪藏请求携带；
  没启用令牌则留空

设置保存在浏览器本地（`chrome.storage.local`），不会同步或上传。

## 常见问题

- **提示「RedInk 未启动（连接失败）」**：本机 RedInk 没有运行或端口不对，
  先启动 RedInk，或在设置里改成实际端口
- **提示「访问未授权」**：RedInk 启用了访问令牌鉴权，请在设置里填写正确令牌
- **提示提取失败**：确认当前是笔记/视频详情页且已加载完成；平台改版时需更新选择器
